## 💎 The Two-File Architecture (Zero-Risk Deployment)

This repository serves as the ultimate template for zero-risk, instantaneous benchmark deployment on any platform (Vercel, Render, or local infrastructure). The entire engine is condensed into exactly **two files**:

1.  **`api/index.py`**: A unified, self-contained logical matrix. It houses the entire FastAPI backend, the automated semantic evaluation judge system, model routing protocols, and the complete React/Tailwind web user interface embedded inside a single script.
2.  **`deps.txt`**: The dedicated environmental blueprint containing every precise package version needed to spin up the gauntlet.

## ⚠️ Critical Deployment Note

> [!WARNING]
> **DO NOT** execute standard environmental installation commands targeting `requirements.txt`. 

```bash

```

In this repository, the `requirements.txt` file is intentionally isolated as a legacy architectural artifact. Attempting to force dependency resolution through it will result in environment conflicts. 

To safely clone and deploy this benchmark, you **must** pass the dedicated dependency manifest to your package manager:


## 🚀 Quickstart

### 🌐 Direct Cloud Access
If you don't want to clone the repository, install dependencies, or run the environment locally, you can directly interact with the running benchmark application here:

[![Launch Benchmark](https://img.shields.io/badge/Launch-AGI_Gauntlet_App-10b981?style=for-the-badge)](https://agibench-2026.onrender.com)

### 💻 Local Setup
1. Clone the repository.
2. Install the verified dependencies:
   ```bash
   pip install -r deps.txt
   ```
3. Set your environment variables:
   ```bash
   export GOOGLE_API_KEY="your-key-here"
   export HF_TOKEN="your-token-here"
   ```
4. Spin up the local FastAPI development server:
   ```bash
   uvicorn api.index:app --reload
   ```

### UI Navigation
Once running, navigate to `http://localhost:8000` to access the **AGI Systems Directorate Terminal**. Select your target inference model (e.g., `aeterna-vox`), assign a secondary model as the Semantic Judge (e.g., `gemini`), and execute the full gauntlet to view live-streamed stability metrics and evaluation logs.
